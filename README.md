# QueryCraft
QueryCraft is a lightweight, intuitive project for managing and exploring data with SQLite. It’s designed to make working with databases simple, organized, and fun—whether you’re testing queries, building tables, or analyzing data.

## Project Highlights
Clean database management – create, read, update, delete records efficiently.

Flexible queries – experiment with filters, joins, and aggregations.

Lightweight and fast – powered by SQLite, no heavy setup required.

Learning by doing – perfect for improving SQL skills while building a functional database project.

## Features
Design and implement custom database tables.

Perform CRUD operations smoothly.

Run SQL queries to explore and analyze data.

Easy to extend for any personal or professional project that needs a database backend.
